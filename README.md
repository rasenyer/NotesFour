# NotesFour
Aplicación de notas desarrollada con Jetpack Compose.

<h3>Características:</h3>
• Guardar nota
<br>
• Editar nota
<br>
• Eliminar nota
<br>
• Eliminar notas

<h3>Librerías usadas:</h3>
• Room
<br>
• Viewmodel
<br>
• Livedata
<br>
• Coroutines
<br>
• Navigation
